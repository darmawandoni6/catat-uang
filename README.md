# catat-uang

## Overview

Aplikasi [Catat Uang](https://catat-uangku.vercel.app/) ini dibuat untuk keperluan pencatatat keuang user dengan tampilan mobile view dan dibisa diinstall di handphone user.

## Feature

- Tracking utang piutang
- Trangking transaksi
- Profil

## Stack yg digunakan

- PWA
- CRA
- Redux
